## 批量导入

::: warning ⚠️

注意这里的输入格式是，必须是英文符号的 `:`，中文符号的 `：` 没办法导入，多个 IP 以换行结尾，如果没有 username 和 password 可以不写

127.0.0.1:7890:username:password

127.0.0.1:7890:username:password

127.0.0.1:7890
:::
